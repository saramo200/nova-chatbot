# 🤖 Nova AI — Bilingual Technical Assistant

A smart, bilingual AI chatbot that responds in **Arabic and English** — powered by OpenRouter AI.

**🌐 Live Demo:** [saramo200.github.io/nova-chatbot](https://saramo200.github.io/nova-chatbot)

---

## ✨ Features

- **Bilingual** — Automatically detects and responds in Arabic or English
- **AI-Powered** — Real AI responses via OpenRouter (not pre-written answers)
- **Chat History** — Remembers the full conversation context
- **Typing Animation** — Smooth loading indicator while AI thinks
- **Distinct Personality** — Nova is professional, warm, and technically focused
- **Secure** — API key stored only in the browser, never on any server
- **Responsive** — Works on desktop and mobile

## 🛠️ Built With

- HTML5, CSS3, Vanilla JavaScript
- OpenRouter API (free tier)
- Llama / Mistral models via OpenRouter

## 🚀 How to Use

1. Visit the [live demo](https://saramo200.github.io/nova-chatbot)
2. Get a free API key from [openrouter.ai](https://openrouter.ai/keys)
3. Enter your key and start chatting!
